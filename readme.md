## 基礎架構

## 主要框架
- Laravel 5.1

## 文件格式
- 編碼：UTF8
- 縮排：4 spaces
- 換行：LF
- 去除行尾空格
- 文件結尾有1空白行

（已使用EditorConfig設定完成）

- PhpStorm

（專案使用[EditorConfig](http://editorconfig.org/)定義基本格式，請安裝與IDE對應的套件）
